These visuals show the fires on our planet. I visualized in visual_world_fires_day.py the world fires 
in one day. Here I used scatter_geo() to generate dots on a world map to show where the fire happened. 
I tried to show through the size and color of dot how bright the fire was. The data for this visual from the 
folder fire_data from the file world_fires_day.csv. 

The data from this file is from the following link:
https://github.com/ehmatthes/pcc_3e/blob/main/chapter_16/mapping_global_datasets/eq_data/world_fires_1_day.csv

I visualized the world fires in a week in visual_world_fires.py. Here I used choropleth instead of geo_scatter() to 
mark the countries and the incedence of their fires in the corresponding colors. The data for this visual is stored in the folder 
fire_data from the file world_fires.csv.

The data is from the following link: https://gwis.jrc.ec.europa.eu/apps/country.profile/downloads
