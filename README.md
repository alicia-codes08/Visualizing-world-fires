This visuals shows the fires of the world. I used visual_world_fires_day.py geo_scatter()
to visualize the world fires and the data from the file world_fires_day.csv. 
The data is from the following link: 
https://github.com/ehmatthes/pcc_3e/blob/main/chapter_16/mapping_global_datasets/eq_data/world_fires_1_day.csv

In the second visual (visual_world_fires.py) I used choropleth instead of geo_scatter() to 
mark the countries and the incedence of their fires in the corresponding colors.
The data is from the following link: https://gwis.jrc.ec.europa.eu/apps/country.profile/downloads
